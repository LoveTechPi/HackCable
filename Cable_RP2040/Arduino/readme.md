# HackCable with Aruduino
We will use Arduino IDE for running ethical hacking codes using C/C++

## Configure and setup the environment
1. Download the [Arduino IDE](https://www.arduino.cc/en/software/) from the official website
2. Setup the RP2040 environemnt after the insallation
   
   `https://github.com/earlephilhower/arduino-pico/releases/download/global/package_rp2040_index.json`

4. Select File > Preference > Additional Board Manager URL and add the above URL
   
<img src="https://github.com/LoveTechPi/HackCable/blob/main/Images/4.png" alt="Image" width="70%"/>

<img src="https://github.com/LoveTechPi/HackCable/blob/main/Images/5.PNG" alt="Image" width="70%"/>

<img src="https://github.com/LoveTechPi/HackCable/blob/main/Images/6.PNG" alt="Image" width="70%"/>


5. Go to the TOOLS->BOARD->BOARD MANAGER. Then search the Pico and install the package

<img src="https://github.com/LoveTechPi/HackCable/blob/main/Images/7.PNG" alt="Image" width="70%"/>

6. Now connect the Hackcable (press the boot button if circuit or micro python installed previously) with the computer
   
7. Go to TOOLS->Boards-> Select the Raspberry Pi Pico
