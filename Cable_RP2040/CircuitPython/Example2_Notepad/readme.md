## Example 2 - Open the notepad and write something on it

This example code will open the Notepad and write a text to it and also blink the RGB LED.

**code.py** - This is the example code, just copy and paste the code to Thonny and run the code.

**adafruit_hid** - Make sure this folder is added to your device

**keyboard_layout_win_uk.py** - This is keyboard layout file, make sure this file is added
