# HackCable with Aruduino
We will use Arduino IDE for running ethical hacking codes using C/C++

## Configure and setup the environment
1. Download the [Arduino IDE](https://www.arduino.cc/en/software/) from the official website
2. Setup the ESP32 environemnt after the insallation
3. Go to the TOOLS->BOARDS->BOARDS MANAGER and install the ESP32
